Demo Brief

Core claim

PatchCore becomes much easier to trust and discuss once the anomaly score is tied to nearest-normal patch provenance and a simple local replacement probe.

Best evidence on this page

The strongest sequence is the selected example view plus the benchmark panel: the report shows the top anomalous patch, the nearest nominal source patch, and the effect of replacing the anomalous region.

How to present it live

Start with one bottle example, show the top patch and nearest-normal exemplar, then use the benchmark panel to show that the same run separates the full test split.

Boundary to say out loud

This report is a strong local PatchCore-style implementation, not a full official benchmark reproduction or a calibrated severity model.

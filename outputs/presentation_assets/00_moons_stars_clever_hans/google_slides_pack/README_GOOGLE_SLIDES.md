# Google Slides Assembly Notes

This folder contains the selected Demo 00 assets for a 10-12 slide presentation. MP4 is preferred over GIF for Google Slides because playback is smoother and file handling is more reliable.

## Upload workflow

1. Upload the `google_slides_pack/` media files or the `mp4/` files to Google Drive.
2. In Google Slides, choose **Insert -> Video -> Google Drive** for MP4 animations.
3. For static figures, choose **Insert -> Image -> Upload from computer**.
4. Use a dark or near-black slide theme with large white section titles and one proof object per slide.
5. Keep the reveal animations large and central; do not build a 23-image gallery.

## Playback settings

- Major reveal animations: **play on click**.
  - `anim_01_moon_moves_confidence.mp4`
  - `anim_02_star_moves_confidence.mp4` if used
  - `anim_11_invisible_background_morph_moon.mp4`
  - `anim_12_invisible_background_morph_star.mp4` if used
- Response-map animations: **play on click** when explaining learned geography.
- Small supporting loops: autoplay only if they do not distract from the speaker.

## Recommended visual treatment

Use a restrained dark theme. Put the claim in the slide title, then use the asset as the proof object. For example: "Same moon. Same pixels. Different place. Different answer." Avoid placing more than two figures on one slide unless the slide is explicitly a comparison.

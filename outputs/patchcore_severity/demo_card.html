Demo 06 - PatchCore Severity Mismatch

Task
Synthetic severity sweep comparing scratch area with PatchCore-style novelty score.
Model
PatchCore-style nearest-neighbour memory bank with colour/texture patch features.
Explanation methods
  • Patch-score anomaly overlay
  • Nearest-normal patch provenance
  • Synthetic severity-mask comparison
  • Severity-rank versus novelty-rank check
Key lesson
A high anomaly score is not automatically a high engineering severity.
Failure mode
Feature-space novelty can rank a small high-contrast mark above a larger low-contrast defect.
Intervention
Add a calibrated severity target and validation layer instead of treating PatchCore distance as severity.
Remaining caveats
  • Synthetic scratch sweep, not a production severity benchmark.
  • No human-labelled severity data yet.
  • No calibrated regression or ordinal severity model yet.

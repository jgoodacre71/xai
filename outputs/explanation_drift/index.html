Demo Brief

Core claim

Prediction drift and explanation drift are different signals, and the explanation can move into the wrong region before headline metrics collapse.

Best evidence on this page

Use the baseline-versus-intervention tables first, then compare attribution-mass columns and anomaly top-patch movement across the perturbation rows.

How to present it live

Pick one nuisance shift, compare the baseline and intervention rows, and then show the corresponding overlays so the audience sees drift rather than only the metric.

Boundary to say out loud

The classifier path uses a prepared NEU-CLS shortcut split in this run, while the anomaly sections depend on whichever local anomaly datasets are prepared on this machine.

Classifier Drift Summary

The baseline is trained on shortcut-correlated real NEU scratches-versus-inclusion shortcut images. The intervention is trained on stamp-randomised and stamp-masked variants of the same parts.

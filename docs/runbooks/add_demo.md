# Runbook: add a new demo

1. Create or update a task file.
2. Confirm the demo's narrative arc:
   - baseline
   - failure or pitfall
   - explanation
   - intervention
   - re-test
3. Decide which of the four XAI questions the demo will answer.
4. Add or extend any shared abstractions needed in `src/`.
5. Add notebook scaffold.
6. Add at least:
   - one unit or integration test
   - one smoke verification path
7. Update the demo catalogue.

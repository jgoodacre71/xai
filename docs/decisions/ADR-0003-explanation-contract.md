# ADR-0003: Shared explanation contract

## Status
Accepted

## Decision
All major demos should attempt to answer:
1. evidence;
2. provenance;
3. counterfactual change;
4. stability.

## Rationale
This creates a common language across classification, anomaly detection, and robustness demos.

## Consequences
- more consistent artefact schemas;
- easier cross-demo comparison;
- more disciplined XAI claims.
